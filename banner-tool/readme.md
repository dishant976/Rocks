\# 🪨 ROCKATERAL — Public API + X Banner Generator



\*\*Live on-chain stats, burned token tracking, wallet personalization, and beautiful X/Twitter banner creation tool\*\* for the ROCKATERAL NFT collection.



Built for the community by the community.



\---



\## ✨ Features



\### Public API

\- Real-time collection stats (supply, burned count, floor price, volume, holders)

\- \*\*Burned tokens tracking\*\* using on-chain `totalSupply` vs `maxSupply`

\- \*\*Wallet holdings endpoint\*\* — Check how many ROCKATERAL any address holds

\- Fast, cached responses with CORS enabled

\- Deployable on Railway / Render



\### X Banner Generator (banner-tool)

\- Live stats dashboard

\- \*\*Wallet personalization\*\* (Option B)

&#x20; - Enter wallet address → See how many Rocks you hold

&#x20; - Enter specific Token ID to feature on banner

\- \*\*6 Planetary Textures\*\*:

&#x20; - Moon Surface

&#x20; - Mars

&#x20; - Earth

&#x20; - Saturn

&#x20; - Jupiter

&#x20; - Deep Space

\- Real-time banner preview (1500×500)

\- One-click PNG download

\- Clean pixel + cyberpunk aesthetic
